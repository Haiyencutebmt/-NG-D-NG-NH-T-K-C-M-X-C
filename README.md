# -NG-D-NG-NH-T-K-C-M-X-C
MoodDiary – Nhật ký cảm xúc là ứng dụng di động đa nền tảng giúp người dùng ghi lại và theo dõi cảm xúc hằng ngày. Ứng dụng có giao diện thân thiện, dễ sử dụng, hỗ trợ thống kê tâm trạng theo ngày và tháng. Đề tài được thực hiện trong học phần Lập trình di động nhằm vận dụng Flutter, Laravel và MySQL vào thực tế.
